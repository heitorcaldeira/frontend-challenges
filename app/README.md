# Desafio Nodis - Frontend

Primeiramente, instale todas as depedências:

```
yarn
```

## Scripts disponíveis:

### `yarn start`

Abre a aplicação em modo de desenvolvimento utilizando o webpack dev server.

Mudanças no código serão refletidas na tela através do hot reload.

### `yarn build`

Faz o build da aplicação para produção na pasta `dist`.

### `yarn test`

Roda todos os testes da aplicação.

### `yarn test:coverage`

Roda todos os testes da aplicação e gera o relatório de cobertura de testes.
